Fitness Tracker App

Overview

This project is a Fitness Tracker App developed as part of the CodeAlpha App Development Internship. 
The application helps users track their daily fitness activities such as workouts, calories burned, and exercise duration.

Features

* Add fitness activity details
* Track workout duration
* Record calories burned
* View daily fitness records
* Store data using browser local storage
* Simple and user-friendly interface

Technologies Used

* HTML
* CSS
* JavaScript

Project Structure

* index.html – Application structure
* style.css – Styling and layout
* script.js – Fitness tracking functionality

How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in any web browser.
